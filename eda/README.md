**Результаты**

2. Bertopic

**Результаты** 
Первоначально Bertopic разделила все тексты на 2 больших кластера, которые имею на самом деле достаточно мало общего между тематиками самого видео
Для достижения лучшего качества тематического моделирования мы уменьшили необходимое минимальное число для вхождения в кластер до 2-х документов.

В результате удалось получить 36 топиков + 1 под аутлаеры. Результат схож с LDA

Эксперименты с регулировкой кол-во текстов для кластера - 16 топиков
