# EDA

Общая база анализа - 333 коротких видео собранных из соцсети Тикток.

Анализ осложняется невозможностью сбора метаданных из парсинга.

Наш текущий анализ разделен на две смысловые компоненты:

- NLP, где мы смотрим семантическое наполнение 

- CV


## NLP

Датасет представляет из себя расшифровку, собранную из 333 видео, с помощью Whisper.
Далее данные поступают в первичный препроцессинг из сырых файлов.

### Конвейер

#### Этап 1. Базовый анализ (Ноутбук test_eda.ipynb)
1. Определение языка с помощью сторонних моделей
2. Фильтрация видео на английском
3. Построение базовых статистик по корпуску
